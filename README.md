# mvn-repo
Maxep's maven repository

### Add Dependency

To add a project hosted on this repo as a dependency, add the following to your project's pom.xml

  ```
    <repositories>
      <repository>
          <id>maxep-releases</id>
          <url>https://raw.github.com/maxep/mvn-repo/snapshots</url>
      </repository>
    </repositories>
  ```
